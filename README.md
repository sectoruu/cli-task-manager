# task-cli

A simple **command-line task manager** built with **Node.js**.  
Manage your tasks directly from the terminal using basic commands.

## Features

- Add tasks
- List all tasks
- Update tasks
- Delete tasks
- Show help guide with `task-cli -h`
- Tasks are stored in a local `JSON` file
- No external libraries

## Tech Stack

- Node.js
- JavaScript
